# Smart Udhar Management System

A digital credit management system for small businesses, kirana stores, medical shops, and farmers to manage customer udhar (credit) efficiently.

## Features
- Customer Management
- Udhar (Credit) Tracking
- Payment Management
- Dashboard with Analytics
- Reports Generation
- Reminder System

## Technology Stack
- PHP 7.4+
- MySQL 5.7+
- Bootstrap 5
- HTML5, CSS3, JavaScript

## Installation

1. Install XAMPP/WAMP server
2. Clone/Copy project to htdocs folder
3. Import database.sql to phpMyAdmin
4. Access: http://localhost/smart-udhar-system

## Default Login
- Username: admin
- Password: admin123

## Database Structure
- 5 main tables with proper relationships
- Support for partial payments
- Transaction history tracking
